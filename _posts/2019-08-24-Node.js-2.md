# dotenv
하드코딩이란 데이터를 코드 내부에 직접 입력하는 것이다. 하지만 코드가 유출되었을 때 중요한 데이터가 노출될 수 있다. 따라서 사용하는 패키지가 dotenv이다.

# 역할
dotenv는 .env파일을 읽어서 process.env 객체에 추가해준다. 따라서 중요한 키들은 .env 파일에 넣어 보호해주면 된다.

# 사용법
```javascript
require('dotenv').config();
```
위처럼 작성해주면 된다.
